# Supabase部署快速开始

## 5分钟快速部署

### 1. 安装Supabase CLI

```bash
npm install -g supabase
```

### 2. 登录并链接项目

```bash
# 登录
supabase login

# 链接项目（替换为您的项目ID）
supabase link --project-ref your-project-ref
```

### 3. 设置密钥

```bash
supabase secrets set GAME_API_CLIENT_ID=your_client_id
supabase secrets set GAME_API_CLIENT_SECRET=your_client_secret
```

### 4. 部署

```bash
# 部署数据库迁移
supabase db push

# 部署Edge Function
supabase functions deploy game-api
```

### 5. 测试

```bash
curl https://your-project-ref.supabase.co/functions/v1/game-api/status
```

## 完成！

现在您的游戏API已经部署到Supabase了！

访问地址：
```
https://your-project-ref.supabase.co/functions/v1/game-api/{endpoint}
```
